# Lessons Index

**Regime:** this file is a **one-line-per-lesson INDEX** — never full prose.
Active rules live in each overlay's Rules section and in `agents/_shared/`
(meta-rules, verify-discipline, browser-access); tier-band changes land in
`agents/pm.md` Rules (single source). Long rationale goes to a dated file
under `agents/archive/`. The Coach owns this index and its consolidation
(see `agents/coach.md`); the PM reads it before constructing briefs.
Rotation: `docs/process-index.md` -> Rotation Regime.

Statuses: `active` (rule lives in an overlay/_shared file) · `folded-M<N>`
(absorbed into a meta-rule) · `superseded-by-LNN` · `obsolete` · `candidate`
(observed, below codification threshold) · `archived` (historical, no live rule).

## Format

```md
- LNN - <status> - <=12-word gist (owning file: agents/<role>.md | _shared/<file>)
```

Every line must be concrete enough to change the next brief, template, or
rule. Lessons are evidence, not vibes — cite lifecycle entries when codifying.
